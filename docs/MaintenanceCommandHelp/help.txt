<fg:yellow>Available commands<:fg>

 * help [command]        - shows this help or help for the specified command
 * monitor               - starts the server monitor
 * show <QueueName>      - starts the queue monitor for the specified queue
 * flush <QueueName>     - flushes the the specified queue
 * flushall              - flushes all queues
 * quit|exit             - ends your maintenance session
